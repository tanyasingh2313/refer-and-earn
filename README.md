# Refer & Earn Web Application

A simple and functional **Refer & Earn system** built with **Node.js, Express, MongoDB, and EJS**.  
Users can register, log in, and generate their unique referral codes to invite others.  
This project demonstrates authentication, routing, database integration, and clean UI flow.
